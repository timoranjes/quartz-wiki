# 律法與恩典

## 定義

在主恢復的職事中，律法與恩典的對比是救恩論和基督徒生活論最重要的課題之一，揭示神對付人的兩種截然不同的原則。[[律法]]是藉著摩西頒布的，包含神的誡命、律例和典章，其功用是顯明罪、定罪罪、並將人圈在罪中，使人認識自己的無能而轉向基督（加三22～23）。律法本身是聖別的、公義的、良善的（羅七12），但它不能賜人生命，也不能使人稱義，它乃是「兒童導師」，引我們到基督那裡。[[恩典]]和實際都是藉著耶穌基督來的（約一17），恩典不僅是神對罪人的赦免和寬容，更是三一神自己——祂的所是、所有和所作——白白賜給信徒，作他們的生命和享受。保羅宣告：「你們不在律法之下，乃在恩典之下」（羅六14），這表明信徒生活的動力不再是律法外面的要求和命令，乃是恩典裡面生命的供應和推動。恩典不僅使人稱義，更將神聖的生命分賜到信徒裡面，產生聖別、變化和模成神兒子形像的功效。因此，律法代表「人應當作什麼」，恩典代表「神已經作成了什麼，並將這成就白白賜給人」。信徒若回到律法之下，就是離開了恩典，失去了[[基督]]作生命的享受；信徒若活在恩典之下，就是活在[[那靈]]的引導中，經歷神聖生命的湧流。

## 比較

| 比較維度 | 律法 | 恩典 |
|----------|------|------|
| **來源** | 藉著摩西頒布（約一17） | 藉著耶穌基督而來（約一17） |
| **本質** | 神聖別的誡命、律例、典章 | 三一神自己作到信徒裡面 |
| **功效** | 顯明罪、定罪罪、不能賜生命 | 赦罪、稱義、賜生命、聖別 |
| **與生命的關係** | 律法不能賜人生命 | 恩典賜永遠的生命 |
| **與稱義的關係** | 沒有人能本於律法稱義 | 稱義是恩典的起點 |
| **與信徒的關係** | 信徒不在律法之下 | 信徒在恩典之下 |
| **生活的動力** | 外面的要求和命令 | 裡面的生命和供應 |
| **結果** | 定罪和死亡（羅七10） | 稱義、聖別和生命（羅五21） |
| **時代** | 舊約時代的主要原則 | 新約時代的主要原則 |
| **神的心意** | 律法是聖別的、公義的、良善的 | 恩典是神心意的終極表達 |

## 職事啟示

倪柝弟兄在《正常的基督徒生活》中清楚指出：「回到律法之下就是離開了恩典，這是加拉太書最嚴厲的警告。」他強調，律法的問題不在於律法本身——律法是聖別的——而在於人沒有能力遵守。律法如同一面鏡子，照出人的污穢，卻不能洗去污穢；它告訴人應當作什麼，卻不能賜人行出來的能力。因此，律法的終極功用是將人引到基督面前，使人放棄靠自己遵守律法的幻想，完全信靠基督。李常受弟兄在《加拉太書生命讀經》中進一步啟示：「恩典不是神對人的善意態度或寬容政策，恩典乃是三一神自己——父在子裡藉著靈——將祂自己和祂所有的一切分賜到信徒裡面，作他們的生命和一切。」他特別強調羅馬書八章二節「生命之靈的律」，指出：「在新約時代，取代摩西律法的不是一套新的規條，乃是生命之靈的律——一個內住的大能，自動地在信徒裡面運行，使他們活出神所喜悅的生活。這遠超過摩西律法的功能。」李弟兄指出，加拉太書是「聖經中的大憲章」，保羅在書中極力抵擋那些要信徒回到律法之下的猶太教信徒，因為一旦回到律法，就與基督隔絕、從恩典中墜落了（加五4）。主恢復的職事一貫堅持：信徒不該追求遵守律法，而該享受基督；不該靠外面的規條來約束自己，而該靠內住的[[那靈]]來生活行事。

## 概念發展

律法與恩典的張力貫穿整本聖經的啟示進程。在舊約中，神藉摩西頒布律法，顯明祂的聖別和公義。但律法不能賜人生命，也不能使人稱義，它乃是引我們到基督那裡去的兒童導師（加三24）。

在新約中，恩典和實際藉著耶穌基督而來（約一17）。恩典不是神對人的善意態度，恩典乃是三一神自己作到信徒裡面作享受。

從神聖啟示的歷史發展：
- **舊約**：律法的頒布，顯明人的無能和神的聖別
- **福音書**：恩典和實際的顯現，基督是恩典的化身
- **保羅書信**：因信稱義，不在律法之下乃在恩典之下
- **主恢復職事**：恩典是三一神的分賜，超越律法的要求

## 實行應用

- **在恩典中生活**：每日倚靠恩典的供應，不是律法的要求
- **享受基督**：以享受基督代替遵守律法
- **生命之靈的律**：操練在生命之靈的律裡生活行事
- **不回到律法**：警惕回到律法主義的試探
- **恩典的傳遞**：以恩典對待別人，不是以律法要求別人

## 關鍵經文

- **約一17**：「因為律法是藉著摩西賜的，恩典和實際都是藉著耶穌基督來的。」
- **羅六14**：「罪必不能作主管轄你們，因你們不在律法之下，乃在恩典之下。」
- **羅八2**：「因為生命之靈的律，在基督耶穌裡已經釋放了我，使我脫離罪和死的律了。」
- **弗二8**：「你們得救是本乎恩，也因著信；這並不是出於你們，乃是神的恩賜。」
- **加三24**：「這樣，律法成了我們的兒童導師，帶我們到基督那裡，使我們本於信得稱義。」
- **來四16**：「所以我們只管坦然無懼地來到施恩的寶座前，為要受憐憫，得恩典，作及時的幫助。」
