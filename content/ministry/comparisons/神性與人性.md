# 神性與人性

## 定義
[[基督]]具有完整的[[神性]]和完整的人性，這兩性在一個位格中不相混亂、不相交換、不能分開、不能離散。基督是神，也是人；祂是神人，是神與人調和的實體。祂在成為肉體時取了人性，卻沒有失去神性。基督的神性使祂能拯救我們到底，祂的人性使祂能體恤我們的軟弱（來四15）。保羅在提前二章5節說：「只有一位神，在神和人中間，只有一位中保，乃是人基督耶穌。」這節經文揭示了基督作為神人兩性的中保職分——祂是神，所以能救贖；祂是人，所以能體恤。
## 比較

| 比較維度 | 神性 | 人性 |
|----------|------|------|
| **源頭** | 神永遠的屬性 | 成為肉身時所取的人性 |
| **在基督裡的地位** | 太初就有的（約一1） | 成為肉體才有的（約一14） |
| **屬性** | 全能、全知、無所不在 | 有限、會飢餓、會疲乏 |
| **與罪的關係** | 與罪無關 | 取了人的樣式，卻沒有罪 |
| **經文中的啟示** | 「話就是神」（約一1） | 「話成了肉體」（約一14） |
| **復活的關係** | 在復活的大能裡被標出（羅一4） | 從大衛後裔生的（羅一3） |
| **救贖的意義** | 有神性才能救贖 | 有人性才能代替人 |
| **中保的角色** | 神與人之間的中保（提前二5） | 祂是神也是人 |
| **與信徒的關係** | 信徒有分於神聖的性情（彼後一4） | 基督在凡事上與我們一樣（來二17） |
| **終極的目標** | 神性與人性在基督裡調和 | 人性被帶進神聖的榮耀裡 |

## 職事啟示
[[李常受]]在《約翰福音生命讀經》中指出：「基督是神，也是人；祂是神人，是神與人調和的實體。」[[倪柝聲]]在《[[正常的基督徒生活]]》中也揭示，基督的人性使祂能體恤我們的軟弱（來四15），祂的神性使祂能拯救我們到底。基督的神性與人性不是簡單的並列，而是調和的——祂在成為肉體時取了人性，卻沒有失去神性。正是因為基督是神人，祂才能在十字架上為我們死，也才能在復活裡將神的生命分賜給我們。[[保羅]]在提前二章5節說：「只有一位神，在神和人中間，只有一位中保，乃是人基督耶穌。」這節經文揭示了基督作為神人兩性的中保職分。
## 概念發展

神性與人性在基督裡的調和是新約啟示的核心真理。

從神聖啟示的歷史發展：
- **約翰福音**：「話就是神」（約一1）啟示神性；「話成了肉體」（約一14）啟示人性
- **羅馬書**：「按肉體說，是從大衛後裔生的；按聖別的靈說，是從死人中復活，以大能標明是神的兒子」（羅一3-4）
- **腓立比書**：「祂本有神的形狀，不以自己與神同等為強奪之珍，反倒倒空自己，取了奴僕的形狀，成為人的樣式」（腓二6-7）
- **提摩太前書**：「只有一位神，在神和人之間，也只有一位中保，就是那人基督耶穌」（提前二5）
- **希伯來書**：祂在凡事上受過試探，與我們一樣，只是沒有罪（來四15）
- **主恢復職事**：強調「神人」的真理，基督是神與人調和的實際，信徒也藉著基督有分於神性

## 實行應用

- **認識基督**：認識基督是完全的神也是完全的人
- **享受神性**：藉著基督，信徒有分於神聖的性情
- **人性被聖別**：人性不是被廢棄，乃是被聖別、被變化
- **活出神人生活**：憑神性而活，在人性中彰顯神
- **中保的依靠**：在軟弱中依靠那位理解我們的中保基督

## 關鍵經文

- **約一1**：「太初有話，話與神同在，話就是神。」
- **約一14**：「話成了肉體，支搭帳幕在我們中間，豐豐富富地有恩典，有實際。我們也見過祂的榮耀，正是從父而來獨生者的榮耀。」
- **羅一3-4**：「論到祂的兒子，我們的主耶穌基督：按肉體說，是從大衛後裔生的；按聖別的靈說，是從死人中復活，以大能標明是神的兒子。」
- **提前二5**：「因為只有一位神，在神和人之間，也只有一位中保，就是那人基督耶穌。」
- **腓二6-7**：「祂本有神的形狀，不以自己與神同等為強奪之珍，反倒倒空自己，取了奴僕的形狀，成為人的樣式。」
- **彼後一4**：「藉這榮耀和美德，祂已將又寶貴又極大的應許賜給我們，叫你們既逃離世上從情慾來的敗壞，就藉這些應許，得有分於神的性情。」
