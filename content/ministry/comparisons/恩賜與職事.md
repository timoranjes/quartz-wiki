# 恩賜與職事

## 定義
[[恩賜]]是聖靈隨己意分給各人的屬靈能力，為要造就召會（林前十二11）；[[職事]]是主所設立的事奉崗位，為要成全聖徒建立基督的身體（弗四12）。恩賜重在聖靈的分賜，職事重在主的設立。恩賜（charismata）是聖靈的運行（energeia），職事（diakonia）是主的安排（diakonia），工作（energēmata）是神的大能。三者源於三一神：聖靈賜下恩賜，主設立職事，神運行一切。保羅在林前十二章將恩賜與身體連結，表明恩賜的價值不在於其本身，而在於是否為著身體的益處。
## 比較

| 比較維度 | 恩賜 | 職事 |
|----------|------|------|
| **定義** | 聖靈的顯現和分賜 | 主的運作和託付 |
| **源頭** | 那靈（林前十二4） | 主耶穌（林前十二5） |
| **性質** | 屬靈的能力和功用 | 服事的崗位和責任 |
| **分給的方式** | 聖靈隨己意分給各人（林前十二11） | 主所安排的服事 |
| **種類** | 智慧的言語、知識的言語、信心、醫病等 | 使徒、申言者、傳福音者、牧人和教師 |
| **目的** | 為著召會的益處（林前十二7） | 為著成全聖徒，建造基督的身體（弗四12） |
| **與三一神的關係** | 出自聖靈 | 出自基督 |
| **在身體中的地位** | 肢體的功用 | 服事的安排 |
| **持續性** | 可能隨環境改變 | 較為持久的託付 |
| **與個人的關係** | 聖靈賜給個人 | 主安排在身體中的服事 |

## 職事啟示
[[李常受]]在《哥林多前書生命讀經》中區分：恩賜是聖靈的運行（charismata），職事是主的安排（diakonia），工作（energēmata）是神的大能。三者源於三一神：聖靈賜下恩賜，主設立職事，神運行一切。[[倪柝聲]]在《[[教會與事奉]]類》中強調，恩賜不等於生命成熟，職事也不等於恩賜的多少。主恢復的目標不是追求恩賜，而是追求生命的長進，使恩賜和職事都能在生命的基礎上服事召會的建造。保羅在林前十二章將恩賜與身體連結，表明恩賜的價值不在於其本身，而在於是否為著身體的益處。
## 概念發展

恩賜與職事的啟示在新約中是多面的。保羅在林前十二章和羅十二章論到恩賜，在弗四章論到職事。

從神聖啟示的歷史發展：
- **羅馬書**：按著所得的恩典，我們有不同的恩賜（羅十二6）
- **哥林多前書**：恩賜原有分別，靈卻是一位；職事也有分別，主卻是一位（林前十二4-6）
- **以弗所書**：基督升上高天，賜下恩賜，設立職事——使徒、申言者、傳福音者、牧人和教師（弗四11-12）
- **彼得前書**：各人要照所得的恩賜彼此服事（彼前四10）
- **主恢復職事**：強調恩賜與職事都為著建造身體，不是為個人榮耀
- **今日實行**：每一位信徒發現並使用恩賜，順服主的職事安排

## 實行應用

- **發現恩賜**：每一位信徒都該發現聖靈所賜的恩賜
- **忠於職事**：在主所安排的職事上盡忠
- **身體的意識**：恩賜和職事都是為著身體，不是為個人
- **配搭服事**：不同恩賜和職事彼此配搭，建造召會
- **不比較**：不比較恩賜的大小，忠心地使用所領受的
- **成全聖徒**：有職事的人成全所有聖徒，使人人盡功用

## 關鍵經文

- **林前十二4-6**：「恩賜原有分別，靈卻是一位；職事也有分別，主卻是一位；功用也有分別，神卻是一位，在眾人裡面運行一切的事。」
- **林前十二11**：「但這一切都是這一位獨一的靈所運行的，隨著祂的意思，個別分給各人。」
- **弗四11-12**：「祂所賜的，有些是使徒，有些是申言者，有些是傳福音者，有些是牧人和教師，為要成全聖徒，目的是為著職事的工作，為著建造基督的身體。」
- **羅十二6**：「按著所賜給我們的恩典，我們得了不同的恩賜。」
- **彼前四10**：「各人要照所得的恩賜，將這恩賜彼此供應，作神諸般恩典的好管家。」
- **林前十二7**：「聖靈顯明在各人身上，乃是為著得益處。」
