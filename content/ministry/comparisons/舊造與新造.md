# 舊造與新造

## 定義
在主恢復的職事中，[[舊造]]與[[新造]]的對比是神永遠計畫的根本分野。[[舊造]]是神起初在亞當裡的創造，包括宇宙、人類和天使，但因撒但的背叛與人的墮落，舊造被罪和死所敗壞，成為伏在虛空和敗壞之下的受造之物。[[新造]]則是神在基督的復活裡所開始的全新創造，不是對舊造的修理或改良，而是從神聖生命產生的全新實體。保羅在林後五章宣告：「若有人在基督裡，他就是新造；舊事已過，看哪，都變成新的了。」[[十字架]]是舊造的終結——不僅是罪的對付，更是整個舊造的了結；[[復活]]是新造的起頭——基督作為復活的首生者，將神聖的生命分賜到信徒裡面。舊造以亞當為源頭，帶著罪性和死的性情；新造以基督為源頭，有分於神聖的生命和性情。信徒的身份轉移不是漸進的改良，而是在基督裡一次永遠的地位改變，需要在日常生活中活出這一新造的事實。

## 比較

| 比較維度 | 舊造 | 新造 |
|----------|------|------|
| **源頭** | 出於亞當的創造 | 出於基督的復活 |
| **本質** | 受造的人帶著罪性 | 有分於神聖性情的人 |
| **與罪的關係** | 被罪捆綁、定罪 | 脫離罪的權勢 |
| **與死的關係** | 死的權勢轄制 | 生命的大能釋放 |
| **終點** | 十字架的終結 | 復活的開始 |
| **神的處理** | 舊造在十字架上被了結 | 新造在復活裡被建立 |
| **生命源頭** | 亞當裡天然的生命 | 基督裡神聖的生命 |
| **地位** | 在神面前被定罪 | 在基督裡被稱義 |
| **生活原則** | 肉體的、屬地的 | 靈的、屬天的 |
| **終極完成** | 舊造要被廢去 | 新造終極完成於新耶路撒冷 |

## 職事啟示
[[倪柝聲]]弟兄在《正常的基督徒生活》中強調一個核心啟示：神對付舊造的路從來不是改良或修理，乃是[[十字架]]的徹底了結；信徒的舊人已經與基督同釘十字架，這不是需要達到的經歷，而是已經完成的客觀事實，信徒需要的是認識並相信這個事實。[[李常受]]弟兄在《以弗所書生命讀經》中進一步開展：新造不僅是個人地位的改變，更是團體性的——神在基督裡創造了[[新人]]，作為神新造的具體彰顯；這個新人由猶太信徒和外邦信徒一同組成，廢掉了規條中的誡命律法，在一個身體裡與神和好。李弟兄指出，新造有分於神聖的生命和性情，經過變化與模成，終極完成於[[新耶路撒冷]]——神與人相互居住的永遠彰顯。這啟示將信徒從個人主義的救恩觀中提升到神團體行動的宏偉異象中。

## 概念發展

舊造與新造的對比是保羅在哥林多後書五章和加拉太書六章的核心啟示：「若有人在基督裡，他就是新造；舊事已過，看哪，都變成新的了」（林後五17）。

在神永遠的計劃中，舊造是神第一次的創造，包括天使、宇宙和人類。但因著撒但的背叛和人的墮落，舊造被敗壞。神在基督裡開始了新造，使所有在基督裡的人都成為新造的一部分。

從神聖啟示的歷史發展：
- **創世記**：神原初的創造，以及第一次的敗壞與恢復
- **福音書**：主耶穌的成為肉體、釘十字架、復活，是新造的開始
- **保羅書信**：在基督裡是新造，舊事已過，都變成新的
- **啟示錄**：新天新地，新耶路撒冷，是神新造的終極完成
- **主恢復職事**：強調「新造」是神聖分賜的結果，不是外面的改良

## 實行應用

- **活在新造裡**：每日確認自己在基督裡是新造的事實
- **不憑舊造生活**：脫離舊造的模式、習慣、思想
- **新造的实际**：在日常生活、人際關係中活出新造的实际
- **新造的眼光**：以新造的眼光看待自己和別人
- **新造的群體**：在召會生活中經歷新造的合一與交通
