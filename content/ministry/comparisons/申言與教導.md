# 申言與教導

## 定義
[[申言]]（prophesying）和[[教導]]（teaching）是召會聚會中兩種不同的功用。申言是將基督說到聽眾裡面，為著建造、勉勵和安慰（林前十四3）；教導是將真理的知識傳遞給信徒，使他們明白神的旨意（太二十八19-20）。保羅說「你們要追求愛，更要切慕屬靈的恩賜，尤其要切慕申言」（林前十四1），表明申言比說方言更優越，因為它直接建造召會。申言不是預言未來，而是將神的話說到人的裡面，使人得著生命的供應。主恢復中提倡的「申言聚會」正是基於這一啟示——每個信徒都應當在聚會中申言，將基督供應給眾人。
## 比較

| 比較維度 | 申言 | 教導 |
|----------|------|------|
| **定義** | 為主說話，把基督說到人裡面 | 將真理和知識傳輸給人 |
| **主要對象** | 人的靈 | 人的心思 |
| **目的** | 建造、勉勵、安慰（林前十四3） | 使人明白真理、認識神 |
| **源頭** | 聖靈直接的感動 | 真理的研讀和領受 |
| **方式** | 即時的、靈裡的說話 | 系統性的、有組織的講授 |
| **在聚會中的地位** | 更優越，為著建造召會 | 基礎性的，為著真理根基 |
| **果效** | 使人得著生命的供應 | 使人得著真理的認識 |
| **與那靈的關係** | 直接從那靈湧流 | 那靈引導的真理傳遞 |
| **需要的操練** | 靈的接觸、對主的敞開 | 真理的學習、知識的累積 |
| **在聖經中的榜樣** | 亞迦布、腓利的四個女兒 | 主耶穌的教導、保羅的教訓 |

## 職事啟示
[[李常受]]在《哥林多前書生命讀經》中強調，申言不是預言未來，而是將神的話說到人的裡面，使人得著生命的供應。[[保羅]]說「你們要追求愛，更要切慕屬靈的恩賜，尤其要切慕申言」（林前十四1），表明申言比說方言更優越，因為它直接建造召會。[[倪柝聲]]在《[[教會與事奉]]類》中指出，教導重在道理的傳授，申言重在生命的分賜。主恢復中提倡的「申言聚會」正是基於這一啟示——每個信徒都應當在聚會中申言，將基督供應給眾人，而不是只有一個人教導、眾人聽。這與傳統教會的講台模式形成鮮明對比，是主恢復在召會生活上的重要恢復。
## 概念發展

申言與教導的區別在新約中有清楚的啟示。保羅在林前十四章特別強調申言的優越性，因為申言直接建造召會。

從神聖啟示的歷史發展：
- **馬太福音**：主的大使命包括「教訓他們遵守」（太二十八19-20），教導的重要性
- **使徒行傳**：安提阿召會中有申言的和教導的（徒十三1），兩類功用並存
- **哥林多前書**：保羅強調「要切慕申言」（林前十四1），申言為著建造
- **以弗所書**：基督賜下使徒、申言者、傳福音者、牧人和教師（弗四11），教導是其中之一
- **主恢復職事**：恢復每一個信徒申言的實行，建立申言的聚會
- **今日實行**：在召會聚會中，鼓勵每一位聖徒申言，不單靠一個人教導

## 實行應用

- **操練申言**：每一個信徒都該操練在聚會中說話，將基督說出來
- **切慕申言**：不滿足於被動聽道，主動追求申言的恩賜
- **教導的根基**：重視真理的教導，建立正確的認識
- **生命的供應**：在申言中供應生命，不單是傳遞道理
- **聚會的建造**：讓每一位信徒都有機會申言，建造全召會

## 關鍵經文

- **林前十四1**：「你們要追求愛，更要切慕屬靈的恩賜，尤其要切慕申言。」
- **林前十四3**：「但那申言的，是對人說建造、勉勵和安慰。」
- **太二十八19-20**：「所以你們要去，使萬民作我的門徒，將他們浸入父、子、聖靈的名裡；凡我所吩咐你們的，無論是什麼，都教訓他們遵守。」
- **徒十三1**：「在安提阿當地的召會中，有幾位申言者和教師，就是巴拿巴和稱為尼結的西面，古利奈人路求，與分封之王希律同養的馬念，並掃羅。」
- **林前十四31**：「因為你們都能一个一个地申言，為要使眾人學教訓，使眾人得勉勵。」
- **帖前五20**：「不要藐視申言者的話。」
