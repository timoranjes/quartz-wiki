# 生命樹與知識樹

## 定義
兩棵樹是創世記二章的啟示，也是整本聖經的鑰匙。[[生命樹]]表徵神自己是人生命的源頭和供應；[[知識樹]]表徵人離開神、憑自己分別善惡的獨立道路。這兩棵樹不是一般意義上的植物，而是兩種生命原則的象徵——生命樹代表倚靠神而活，知識樹代表獨立於神之外。整本聖經從創世記的生命樹開始，到啟示錄的生命樹結束（啟二十二2），表明神永遠的定旨是要人吃祂、接受祂、以祂為生命。人墮落的根本原因不是道德的失敗，而是離開了生命樹、吃了知識樹。
## 比較

| 比較維度 | 生命樹 | 知識樹 |
|----------|--------|--------|
| **表徵** | 神自己作人的生命 | 撒但作死亡的來源 |
| **食物的性質** | 生命的供應 | 死亡的毒素 |
| **神的命令** | 園中各樣樹上的果子你都可以吃 | 分別善惡樹上的果子你不可吃 |
| **人的選擇** | 人應當選擇吃生命樹 | 人選擇吃了知識樹 |
| **結果** | 生命、永遠的福分 | 死亡、墮落、敗壞 |
| **原則** | 依靠神、接受神 | 獨立於神、自己決定善惡 |
| **與神的關係** | 以神為源頭 | 離開神、背叛神 |
| **在聖經中的發展** | 創二到啟二十二，貫穿全聖經 | 創三到啟二十，最終被扔入火湖 |
| **在新約的應驗** | 基督是生命樹的實際 | 律法是善惡知識的延續 |
| **終極結局** | 新耶路撒冷中的生命樹（啟二十二） | 被永遠消滅 |

## 職事啟示
[[李常受]]在《創世記生命讀經》中強調：「神不要我們作善惡的分辨者，神要我們作生命的享受者。」整本聖經從創世記的生命樹開始，到啟示錄的生命樹結束（啟二十二2），表明神永遠的定旨是要人吃祂、接受祂、以祂為生命。[[倪柝聲]]在《[[正常的基督徒生活]]》中也指出，人墮落的根本原因不是道德的失敗，而是離開了生命樹、吃了知識樹。因此，神恢復的工作就是把人帶回生命樹——不是靠人的努力行善，而是靠接受神作生命。主恢復中「生命」與「知識」的區別正是基於這兩棵樹的啟示。
## 概念發展

生命樹與知識樹的啟示貫穿整本聖經，是聖經的中心思想。

從神聖啟示的歷史發展：
- **創世記**：兩棵樹放置在伊甸園中，人類選擇了知識樹（創二9，三6）
- **箴言**：「她（智慧）是生命樹」（箴三18），指向生命樹的原則
- **約翰福音**：基督是生命的糧（約六35），是生命樹的實際
- **羅馬書**：律法是善惡知識的延續，生命是藉著基督（羅七，八）
- **啟示錄**：生命樹在新耶路撒冷中恢復（啟二十二1-2，14）
- **主恢復職事**：強調「吃基督」，恢復生命樹的經歷

## 實行應用

- **選擇生命樹**：每天選擇基督作生命，不憑善惡標準生活
- **吃生命樹**：藉著禱讀、呼求、享受基督，實際地吃生命樹
- **拒絕知識樹**：不憑自己的善惡判斷，不留在批評、論斷中
- **生命的供應**：在聚會中供應生命，不傳遞知識
- **恢復伊甸園**：在新耶路撒冷中，生命樹永遠恢復

## 關鍵經文

- **創二9**：「耶和華神使各樣的樹從地裡長起來，能悅人的眼目，也好作食物；園子當中有生命樹，還有善惡知識樹。」
- **創二17**：「只是分別善惡樹上的果子，你不可吃，因為你吃的日子必定死。」
- **創三6**：「於是女人見那棵樹的果子好作食物，也悅人的眼目，且是可喜悅的，能使人有智慧，就摘下果子來吃了，又給她丈夫，她丈夫也吃了。」
- **啟二十二1-2**：「天使又指示我一條生命水的河，明亮如水晶，從神和羔羊的寶座流出來。在河這邊與那邊有生命樹，生產十二樣果子，每月都結出果子，樹上的葉子乃為醫治萬民。」
- **啟二十二14**：「那些洗淨自己袍子的有福了，可得權柄到生命樹那裡，也能從門進城。」
- **約六35**：「耶穌對他們說，我就是生命的糧，到我這裡來的，必定不餓；信入我的，永遠不渴。」
